# pathmakers-semesterA
 project by May&Eden


## אודות הפרויקט
PathMakers הוא אתר אינטראקטיבי המסייע למשתמשים בתכנון מסלולי טיול מותאמים אישית. המערכת עושה שימוש בבינה מלאכותית כדי להציע מסלולים מבוססי העדפות המשתמש, מגבלות ונתוני יעד.





### **Frontend (React)**

│-- AboutPage.jsx
